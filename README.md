## Custom Rules (Experimental)

Rules implemented in this scanner that are not drawn from any external rule set. These are **not included in the default rule set** due to higher false-positive rates. To use them, import and pass them explicitly:

```js
import { Scanner } from "@koddsson/accessibility-scanner";
import errorMessage from "@koddsson/accessibility-scanner/rules/error-message";

const scanner = new Scanner([...allRules, errorMessage]);
```
